1) бот должен сидеть в чате
2) бот должен вызываться по команде через собаку как юзернейм
3) после вызова бота например @bot показывать список доступных команд в нативном тг окне
    доступные команды:
    - list_stands
    - list_free_stands
    - claim
    - ping
    - release

/list_stands - команда отображает список всех доступных стендов в прицнипе
/list_free_stands - команда отображает спискок свободных для клейма стендов
/claim - забрать себе стенд, при этом засетив в базе free: false
/ping - пингануть типа по юзернейму, чтобы отдал стенд. Пингануть можно только того типа, за которым закреплен стенд
/release - отпустить стенд, при этом должен сработать колбек, который отправит в чат сообщение типа (@username has released {standName})
